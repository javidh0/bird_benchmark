Key Findings
✅
RESULT New Agent beats Old Agent on every dataset and both models
Corrected accuracy: OpenAI New 59.2% vs Old 41.9% (+17.3pp). Qwen New 51.0% vs Old 40.5% (+10.5pp). The improvement holds across all 4 datasets without exception. Biggest single gain: OpenAI on Superhero (+29.5pp).
⚡
SPEED New Agent is ~2.2× faster than Old Agent on both models
OpenAI: New 13.1s vs Old 28.4s (−15.3s per query). Qwen: New 21.3s vs Old 45.9s (−24.6s per query). Speed improvement is consistent across all 4 datasets — not limited to any specific domain.
🔴
RISK OpenAI has 3× more schema read failures than Qwen
OpenAI returned completely empty SQL 283 times vs Qwen's 93. When the agent can't read the schema it produces nothing at all. Qwen is notably more resilient on complex or unfamiliar schemas.
🔍
FIX APPLIED 49 column order mismatches were false failures
28 OpenAI + 21 Qwen failures had correct data but wrong column order. These are treated as passes in the corrected accuracy figures shown above — OpenAI gained +4.9pp and Qwen +3.6pp from this correction. The evaluator should be made order-insensitive.
📊
FINDING Superhero is the strongest dataset; Formula 1 the hardest
Corrected: Superhero hits 72% on both models. Formula 1 is weakest (51–57%) due to complex multi-table racing queries requiring time-based filtering and nested joins. Both models struggle equally there.
🟡
FIX Qwen returns zero rows 2× more than OpenAI
Qwen generates valid SQL but uses incorrect filter values 175 times vs OpenAI's 87. The agent selects the right table and column but uses the wrong string literal. Domain-specific vocabulary (exact enum values) is a known weak point for Qwen.
📋 Accuracy methodology note: All accuracy figures shown above are corrected — 49 column order mismatches (28 OpenAI + 21 Qwen) where correct data was returned in the wrong column order are treated as passes, as these were evaluator false-negatives rather than agent errors.
Raw (pre-correction) accuracy for reference — OpenAI New: 54.3% overall (Cal 33.7%, F1 49.4%, SH 70.5%, CB 57.5%) · OpenAI Old: 36.9% overall  |  Qwen New: 47.4% overall (Cal 28.1%, F1 44.8%, SH 71.3%, CB 42.5%) · Qwen Old: 37.0% overall
